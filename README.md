# 🚨 Bike Security DApp (SOS Alert System)

## 📌 Overview

Bike Security DApp is a blockchain-powered vehicle safety application designed to provide emergency SOS alerts during accidents.
The system securely stores alert information on the blockchain using MetaMask authentication and smart contract integration.

This project combines Web3 technology, blockchain security, and real-time emergency support into a modern decentralized application.

---

# ✨ Key Features

* 🚴 Accident / SOS alert triggering
* 🔐 Secure blockchain transaction storage
* 🦊 MetaMask wallet integration
* 📍 Google Maps accident location display
* 📧 Emergency email notification system
* 📊 Sensor activity history dashboard
* 🌐 Modern responsive user interface

# 🔄 Workflow

1. User registers and logs into the application
2. MetaMask wallet is connected
3. SOS sensor is triggered during emergency
4. Blockchain transaction confirmation occurs
5. Emergency alert email is sent
6. Accident location is displayed on Google Maps
7. Sensor activity gets stored securely

# 🚀 How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/dApp-sos.git
```

---

## 2️⃣ Open Project Folder

```bash
cd dApp-sos
```

---

## 3️⃣ Install Dependencies

Make sure MetaMask extension is installed in your browser.

---

## 4️⃣ Start Ganache

Run Ganache blockchain locally.

---

## 5️⃣ Open Frontend

Open `index.html` in your browser.

---

## 6️⃣ Connect MetaMask

* Import Ganache account
* Connect wallet
* Approve transactions
