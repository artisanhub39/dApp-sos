# 🚨 Bike Security DApp (SOS Alert System)

## 📌 Overview

Bike Security DApp is a blockchain-powered emergency vehicle safety system designed to provide instant SOS alerts during accidents.
The application securely stores emergency records on the blockchain using Web3 technology and MetaMask authentication.

This project combines blockchain security, decentralized applications, emergency response systems, and real-time monitoring into a modern Web3 solution.

# 🔄 Application Workflow

1. User registers and logs into the system
2. MetaMask wallet is connected
3. Emergency SOS sensor is triggered
4. Blockchain transaction is initiated
5. MetaMask confirms the transaction
6. Accident location is displayed on Google Maps
7. Emergency email alert is sent
8. Sensor activity gets stored securely
