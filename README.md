# 🚨 Bike Security DApp (SOS Alert System)

## 📌 Description
A blockchain-based vehicle safety system that triggers an SOS alert during accidents and stores data securely on the blockchain.

---

## ⚙️ Tech Stack
- Python  
- HTML, CSS, JavaScript  
- Ganache / Polygon  
- MetaMask  

---

## 📊 Architecture
User → Web App → Python Backend → Blockchain → MetaMask  

---

## 🔄 Workflow
- User triggers SOS  
- Backend processes request  
- MetaMask confirms transaction  
- Data stored on blockchain  

---

## 🚀 Features
- Accident/SOS detection  
- Blockchain data storage  
- Secure transactions using MetaMask  

---

## 🛠️ How to Run
1. Clone the repository  
2. Install required dependencies  
3. Run the backend (Python)  
4. Open frontend in browser  
5. Connect MetaMask  

---
